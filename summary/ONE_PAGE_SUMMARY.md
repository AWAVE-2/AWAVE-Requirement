# AWAVE - One Page Summary

## The Product
**AWAVE** is a meditation and sound therapy mobile app (iOS/React Native) that helps users achieve relaxation, focus, and better sleep through procedurally generated audio and ambient soundscapes.

## Core Value Proposition
- Multi-track audio mixing with real-time customization
- Procedural sound generation (frequencies, ambient, nature sounds)
- Personalized wellness experiences with usage analytics
- Offline-capable with session-based downloads
